# Mobile par Kaise Chalayein? (Step-by-Step Guide)

Aap is tool ko apne mobile par **2 aasaan tareeko** se chala sakte hain:

### Tareeka 1: WhatsApp ya Telegram (Sabse Aasaan)
1. Apne computer mein `/Users/mac/Documents/flutterapps/WIN GO/` folder kholein.
2. **index.html** file ko copy karein aur apne aap ko **WhatsApp**, **Telegram**, ya **Email** par bhej dein.
3. Apne mobile mein us file par click karein aur use **Chrome** ya **Safari** browser mein open karein.

### Tareeka 2: Local Wi-Fi Server (Best for 30s Game)
Agar aapka Mac aur Mobile ek hi Wi-Fi par hain:
1. Mac ke terminal mein ye command chalayein:
   `cd "/Users/mac/Documents/flutterapps/WIN GO" && python3 -m http.server 8080`
2. Apne Mac ka IP address check karein (Settings -> Wi-Fi -> Details). Man lijiye IP `192.168.1.5` hai.
3. Apne mobile browser mein ye address dalein: `http://192.168.1.5:8080`

---

### Mobile Par "App" Kaise Banayein?
Jab aapka tool browser (Chrome/Safari) mein khul jaye:

- **iPhone (Safari):** Neeche **Share** button pe click karein -> **"Add to Home Screen"** select karein.
- **Android (Chrome):** Upar 3-dots menu pe click karein -> **"Install App"** ya **"Add to Home Screen"** select karein.

**Fayda:** Iske baad ye aapke mobile mein ek **Real App** ki tarah icon ban jayega aur aap ise turant open kar payenge.

---

### Game Ke Saath Kaise Use Karein?
1. Ek phone mein **Jalwa Site** kholein.
2. Dusre phone mein (ya split screen mein) ye **Predictor Tool** kholein.
3. Jaise hi result aaye, turant tap karein aur agla prediction dekhein.
